# Digital World Portal

A simple single-page website with a fixed layout:

- **Top (10%)** — Welcome header
- **Left (20%)** — Menu: Sign-in, Sign-up, About
- **Right (80%)** — Detail content (default: Welcome to Digital World)

## Usage

Open `index.html` in your browser, or host it on GitHub Pages.

## GitHub Pages

1. Push this repository to GitHub.
2. Go to **Settings → Pages**.
3. Select the branch and `/ (root)` folder.
4. Save and open the published URL.
